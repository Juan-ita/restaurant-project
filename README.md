# Professional Company Page

## Golden Bites
A professional restaurant landing page built using HTML5 and deployed on git hub.

------------------
## Project Description

The goal of my project was to build a professional landing page using HTML5 structure while practicing Git workflow, accessibility and deployment.

------------------

## Motivation
The motivation behind this project was to practice building a real-world business website.

Also to improve my understanding of HTML structure, GitHub workflow and project deployment.

## Reason as to why I built this project
I built this project to simulate a professional client request for a restaurant website and strengthen my practical web development skills.

------
## What i learned
Through this project i learned how to:

- Structure websites using HTML
- Create navigation menus
- Use Git and GitHub professionaly
- Push changes and manage commits

-------

## Features
* Hero Section
* Menu Section
* About Section
* Contact Section
* Embedded Images
* Footer with social links
* Customer Review Section
